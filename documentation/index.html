TypeORM で マイグレーション (初回)

xxx.entity.ts でモデル定義

```bash

マイグレーションファイル作成

yarn ts-node $(npm bin)/typeorm migration:generate -n Initialize

マイグレーション 実行

yarn ts-node $(npm bin)/typeorm migration:run

マイグレーション 戻し

yarn ts-node $(npm bin)/typeorm migration:revert ```

docker memo

```bash

初期

docker pull mysql:5.7.24 docker run -d --name mysql -e MYSQL_ROOT_PASSWORD=mysql -e MYSQL_DATABASE=task -p 3306:3306 mysql:5.7.24 --character-set-server=utf8 --collation-server=utf8_unicode_ci

開始

docker start mysql

停止

docker stop mysql

対話形式

docker exec -it mysql bash mysql -uroot -p mysql

mac から

mysql -uroot -p -h 127.0.0.1 ```
